# Vim Simple Bookmarks

## Installation

### Plug
`Plug 'maxboisvert/vim-simple-bookmarks'`

Or use your prefered way.

## Usage

### Add a bookmark
```
call AddBookmark()
```

### List bookmarks
```
call ListBookmarks()
```

This will open the bookmarks list. Press enter to go to a bookmark.

### Delete bookmarks

press `dd` when a bookmark is selected in the list.

```
call ClearBookmarks()
```
